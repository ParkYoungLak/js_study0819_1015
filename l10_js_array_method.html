자바스크립트 배열의 유용한 메소드

const a=["b","c","a","z","f","e","h"];

  • a.toString(): 쉼표로 구분된 배열의 값을 문자열로 출력한다.
  • a.join(): 배열의 값을 join 안에 문자열로 결합한 문자열을 출력한다.
  • a.pop() : 배열의 마지막 값을 반환한다. 원배열의 값이 변화함.
  • a.push() : 배욜의 마지막에 새 값을 추가한다. 원배열의 값이 변화.
  • a.shift() : 배욜의 첫번째 값을 반환하고 제거한다.. 원배열의 값이 변화.
  • a.unshift() : 배욜의 첫번째에 새 값을 추가한다. 원배열의 값이 변화.
  • a.concat([1,2,3]): 배열의 끝네 새로운 배열을 추가한 배열을 반환한다. 원배열 변환 x
  • a.slice(start_index,end_index) : 시작점부터 끝점까지의 배열을 반환한다.end_index제외
  • a.splice(start_index,n(지울갯수),insert_item1(추가요소.)... ) : arr[시작점+1]부터 n개만큼 삭제값을 반환.원배열 변환..
  • a.sort() : 배열의 요소를 순서대로 정렬만 한다.
  • a.reverse() : 배열의 요소를 역순으로 정렬만 한다.
