숫자와 부동 소수점을 포함하는 Number type

numbersms 64비트의 부동 소수점인 숫자를 저장하는 기본 데이터 타입.

  • number가 표현할 수 있는 최대 수(max_safe_integer)
  • number가 표현할 수 있는 최소 수(min_safe_integer)
  • javascript가 표현할 수 있는 가장 큰 수
  • javascript가 표현할 수 있는 가장 작은 수
  • Number. 는 정수와 소수를 표현한 데이터 타입을 javascript가 브라우저 동작시 생성한 객체.
  • number 는 총 64bit를 저장하고 분수는 0~51, 지수는 52~62 부호(.)이 63에 저장된다.

number의 연산과 자동형 변환

숫자는 연산을 하기 위해 존재하고 javascript의 숫자는 숫자가 아닌 것도 연산하기 위한 형변환을 시도한다.

+연산은 문자열을 합치는 연산이기도 하다. 때문에 숫자의 +연산시 문자열이 있다면 숫자를 문자열로 형 변환한다.

  • "A"+10+20 :
  • 10+20+"A" :
  • 1+false :

+연산을 제외한 나머지 연산은 문자열이 나올 시 자동 형 변환을 시도한다. 만약 형변환되지 않으면 숫자가 아니라는 뜻의 NaN이 된다. false와 null 은 0으로 변환되고, NaN과 undefined는 NaN이 된다.
