자바스크립트의 데이터 타입과 기본 데이터 타입

자바스크립트의 변수는 모든 데이터 타입을 참조할 수 있다.(동적타입 dynamically typed) 이 때 모든 프로그래밍 언어와 같이 자주 쓰이는 타입을 기본데이터 타입으로 지정하고 있으며 총8가지가 있다.

typeof Number 숫자형

정수와 (float)부동소수점, 무한대(Infinity), NaN(not a number)숫자가 아닌 것에 대한 오류표기. 숫자형은 *,+,-,/ 등을 연산을 할 수 있다.

  • 1/0 :
  • -1/0 :
  • "십"/0 :
  • "10"/2 : (자동으로 형을 변환한다. "10"->10)
  • " 10 "/2 : (자동으로 형을 변환한다. 공백포함" 10 "->10)
  • typeof : 데이터 타입을 확인할 수 있다.
  • let x =100; typeof x
  • x= 3.14;
  • x= "십"; type of x=
