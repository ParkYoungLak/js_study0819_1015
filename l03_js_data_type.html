자바스크립트의 데이터 타입과 기본 데이터 타입

자바스크립트의 변수는 모든 데이터 타입을 참조할 수 있다.(동적타입 dynamically typed) 이 때 모든 프로그래밍 언어와 같이 자주 쓰이는 타입을 기본데이터 타입으로 지정하고 있으며 총7가지가 있다. (string, number, bigint, boolean, symbol, null, undefined)

기본데이터 타입은 아니지만 기본데이터 타입처럼 사용할 수 있는 배열과 JSON

let arr = [a,b,c,d]; new 없이 생성가능 (arr = new Array(a,b,c,d)과 동일)

배열은 순서를 가지는 데이터타입으로 모든 데이터 타입을 참조할 수 있다.(같은 데이터 타입을 참조하는 것을 권장)

let json = {key:value,key2:value2} : new 없이 생성가능한 JSON 타입(new object())

json 은 key와 value 를 가지는 데이터 타입으로 new Object 로 생성가능. 이때의 Obj는 모든 클래스의 부모인 Object와 별개의 클래스이다.
