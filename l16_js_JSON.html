JSON(Javascript Object Notation)

  • 자바스크립트의 객체 표현법
  • JSON 은 문자열로 표현될 수 있다.
  • 아주 가벼운 경량 데이터 교환 방식으로 문자열로만 이루어진다.(모든 프로그래밍 언어에 독립적이다.)
  • 데이터를 주고 받기위해 사용된다.

JSON 작성법

  • JSON = {}
  • const jsonData = {};
  • const jsonData = "{key:value, key2:value}"; 이때 KEY의 이름은 중복될 수 없다.
  • const jsonData = '{\"key\":\"value\", \"key2\":\"value\"}';
  • 문자열로 이루어진 데이터 형식이기 때문에 기본데이터 타입(문자열,숫자,논리,null)과 배열을 제외하고 모두 문자열로 작성.
  • JSON 형식의 문자열을 자바스크립트에서 JSON.parse 로 object type 으로 변경한다.
  • object 로 변경된 json 의 key 는 property 로 .으로 접근가능하다. : jsonData.key
  • 프로그래밍에서 key 가 유동적으로 바뀔 때는 ["key"] 방식으로 접근가능하다. : jsonData["key"]

자바스크립트가 JSON을 대하는 태도.

  • 자바스크립트에서 JSON 은 object() 의 프로퍼티 설명서.
  • new Object() 로 JSON 을 생성할 수 있다.
  • 자바스크립트에서 JSON 을 기본데이터 타입처럼 리터럴한 방식을 제공한다.
  • {key:valeu} : 리터럴방식(자바스크립트 문법에서 사용되는 JSON )
  • '{"key": "value"}' : 문자열로 표현된 JSON (서버에서 주고받는 데이터)
  • 자바스크립트는 함수지향 언어로 함수를 객체로 선언할 수 있다.
  • function person(name,age){this.name=name, this.age=age => new person("최경민",36)= person : {name : "최경민", age : 36}
  • 자바스크립트 객체의 프로퍼티 설명서가 문자열로 데이터 전송되는 것을 JSON 이라고 한다.
