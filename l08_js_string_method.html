자바스크립트 문자열 String의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재한다.

method 설명 예시
toUpperCase(),toLowerCase() 문자열의 대문자 or 소문자 변환 "Ab Cd".toUpperCase() : "Ab Cd".toLowerCase() :
trim() 문자열의 앞뒤 공백으 제거 "최경민 ".trim().length :
charAt(),charCodeAt() "ABCDE".charCodeAt(0)
"ABCDE".charAt(0) "ABCDE"[0]
slice(start,end) 시작점은 포함,엔드는 제외 일부 문자를 추출할 때 사용. "스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
substring(start,end) 시작점은 포함,엔드는 제외 slice와 동일하지만 양수 인덱스만 가능.. "스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
substr(start,length) 시작점제외 일부 문자를 추출.이때 시작점부터 길이까지 추출한다. "스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
"스파게티 피자 된장찌개 샌드위치 "
startsWith() 특정문자열로 시작하는지 여부. foodList.startsWith("스") : foodList.startsWith("된장",8) :
endsWith(String,position=String.length) 끝에서부터 특정문자열로 끝나는지 여부.boolean으로 판별. foodList.endsWith("위치") : foodList.endsWith("샌드위치") : foodList.endsWith("된장",10) :
includes(String,start position=0) 특정 위치부터 특정 문자열이 있는지 여부,boolean으로 판별. foodList.includes("찌개"); foodList.includes("찌개",12);
indexOf(String,start position=0) 특정 위치부터 특정 문자열이 있는지 검색 후 시작지점(index)반화, 없으면 -1반환 foodList.indexOf("피자 ") foodList.indexOf("피자 ")
lastIndexOf(String,start position=0) 특정 위치부터 역방향으로, 검색 후 시작지점(index)반환, 없으면 -1반환 indexOf는 중복시 처음것 반환, lastIndexOf는 마지막 것 반환. foodList.lastIndexOf("Brave") foodList.lastIndexOf("Brave")
foodList.indexOf("Brave")
foodList.indexOf("new")
split(String,limit length) 특정 문자열을 기준으로 나누어 배열로 만든다.(길이를 정할 수 있다.)
foodList.split(",")
replace() 특정 문자열이나 정규식 패턴으로 검색 후 다른 문자열로 변환한 값을 반환 원래 배열의 데이터가 바뀌는게 아님.검색된 값 1개만 바꿈. foodList.replace("된장찌개","스테이크") foodList.replace(" ",",") foodList.replace("된장찌개","$& 제외 스테이크 추가")
replaceAll() 특정 문자열이나 정규식 패턴으로 검색한 모든 문자열을 다른 문자열로 변환한 값을 반환. foodList.replaceAll(" ",",");
