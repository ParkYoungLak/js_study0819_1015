자바스트립트의 변수

모든 프로그래밍 언어는 데이터를 참조하기 위해 변수를 사용한다. 이 때 참조하는 데이터를 변할 수 있어서 변수다.(let a="안녕"-> a=200)

자바스크립트의 변수와 상수

  • var : 자바스크립트의 태초의 변수로 함수 지향언어의 특징을 가진 블럭이 모호한 상태다.
  • let : ES6에 탄생한 변수로 클래스 추가와 함께 블럭의 구분을 명확하게 한다.(var 대신 사용)
  • const : 상수. 선언과 동시에 데이터를 참조해야하고 참조된 데이터를 변경할 수 없다.

변수의 식별자(이름)

프로그래밍에서 식별자 즉 이름 짓기는 매우 중요하다.대체로 유지보수에 용의하기 위해 약속된 방식이지만 이름이 오류를 만들 수 있어 오류가 있는 코드를 배포하는 자바스크립트 특성상 이름 짓기가 더 중요.
예) let var=10; , let function="test"; //예약어 오류
function a(){let a;} //권장하지 않음.

  • 변수명은 문자,숫자,underbar(_), $ 가 포함될 수 있다.
  • 변수명은 숫자로 시작하는 것을 권장하지 않는다. 예) let 1num;(x) let num1;(o)
  • 변수로 이름은 구체적이야하고 문자와 문자의 결합을 카멜포기법(낙타)or _로 구 예) javaScript, memNum, userId(카멜 표기법) java_script, mem_num, user_id(_표기)
  • 상수(const)의 이름은 모두 대문자이고 _표기한다.(color_red)
  • 이름은 대소문자를 구분한다.(Num!=num!=NUM 모두 다른 변수)
  • 예약어는 변수 이름으로 사용할 수 없다.(function, var, let, Number...)

식별자의 그 밖의 규칙(권장)

  • 읽고 이해할 수 있는 구체적인 이름을 사용한다.(userName, colorName, colorHexCode)
  • a,b,c, i, num, val 같은 짧은 변수명은 피한다. 구조적으로 변수명이 명확할 떄 사용가능. 예) for i->index, mem(num)
  • 소속된 팀의 규칙을 따른다.(userId, memberNum)x, (userId, userNum)o
