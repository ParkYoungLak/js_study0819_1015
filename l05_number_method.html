Number 객체의 전역 변수와 함수

기본데이터 타입인 number는 Number라는 타입의 객체를 가진다. 이떄 기본데이터 타입을 보조하기 위해 Number가 내장 객체(new 연산자없이 사용할 수 있는 객체) 로 존재하며 다양한 전역변수(어디든 사용할 수 있는)와 메소드를 제공하고 있다.

내장객체 Number의 전역변수

변수명(Property) 설명
Number.MAX_VALUE 넘버가 표현 가능한 가장 큰 수(e308 : 10의 308승 까지 표현가능 )
Number.MIN_VALUE 넘버가 표현 가능한 가장 소수점이 낮은 수(-e324 : 10의 324승 분의 1 까지 표현가능 )
Number.NaN 숫자가 아님의 값을 검사(하지만 Number type),수가 아닌 것을 연산할 때 예) "십"*10
Number.MAX_SAFE_INTEGER 넘버가 표현 가능한 가장 큰 정수(짤리지 않는 수)
Number.MIN_SAFE_INTEGER 넘버가 표현 가능한 가장 작은 정수(짤리지 않는 수)

Number 내장객체의 메소드

메소드 설명
Number.isNaN() 전달된 값이 NaN인지 검사. 숫자가 아니면 (NaN)이면 true 반환. 예1)Number.isNaN("")
Number.isFinite 전달된 값이 유한 숫자인지 검사(유한:true / 무한:false) 예1)Number.isFinite(10/0) 예2)Number.isFinite(10/2)
Number.isSafeInteger 안전한 정수인지 검사 예1)9e+20: 예2)9e+15:
Number.
Number.isInteger 전달된 값이 정수인지 판단(정수:true, 소수,NaN,Infinity,boolean:false) 예1)Number.isInteger(Math.PI) 예2)Number.isInteger(-222) 예3)Number.isInteger(-222.0)
Number.parseFloat() 문자열의 부동 소수를 찾아서 소수점 숫자를 반환.(공백으 포함한 문자열 앞에 소수를 찾아서 부동소수로 반환) 예1)Number.parseFloat("3.14abcdefskdfj") : 예1)Number.parseFloat("alksd3.14") :
Number.parseInt() 문자열의 정수를 찾아서 반환 예1)Number.parseInt() : 예2)Number.parseint("alksd3.14") :
