Element.classList : html 배열타입으로 자바스크립트 배열과는 다른 함수를 제공한다.

요소는 복수의 클래스를 가질 수 있다.

Element.class = "a b c"(사용안함)

Element.classList = [a, b, c] (o)

스타일이 적용된 클래스를 추가 삭제하기 위해 많이 사용된다.

  • classList.add("ClassName"): array.push() 비슷한 함수로 클래스를 추가한다.
  • classListr.emove("ClassName"): array.slice() 비슷한 함수로 클래스를 제거한다.
  • classList.toggle("ClassName"): 해당클래스가 있으면 삭제 없으면 추가.
