자바스크립트의 조건문

블럭안의 실행을 조건에 따라 동적으로 선택되게 하는 조건문.if, switch, 삼항연산자.

if 조건문

if(condition){실행} : 조건이 참(true)이면 블럭을 실행한다.

if(){}else{} :if 의 조건이 맞지 않으몀ㄴ else의 블럭을 실행한다.

if(조건1){실행}else if(조건2){실행}else{실 : 1번조건이 만족하지 않으면 2번조건 실행 모두 만족하지 않으면 else 이하를 실행.}

  • if(value < 50){}
  • 50보다크거나 20보다 작은수?

  • 10~20 || 30~40

Switch 조건문

            switch (변수){
              case a: //(변수===a)
              실행
                break;
              case b:
              실행
                break;
                case c:
                실행
                  break;
                  default;
                  실행
               }

스위치 케이스는 엄격한 동등비교(===)를 사용하고, 위에서부터 위치하는 항목을 찾으면 switch는 종료된다. (엄격한 비교 외의 조건을 사용할 수 없다.)

if에서 else if문과 동일하게 작동하지만 조건문의 수행 속도가 굉장히 빠르다.

조건을 파악하기 굉장히 좋다.
