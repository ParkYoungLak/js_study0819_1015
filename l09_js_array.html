자바스크립트의 배열

여러 변수를 index로 참조해 저장하는 데이터 타입(literator클래스를 상속하고 있다).

작성법: const array=[item1,item2,item3,...];(권장.기본데이터타입처럼 사용가능.)

작성법2: const array= new Array(item1,item2,item3,...); (권장하지 않음.)

호출법: array[index]

array.toString(): 객체를 호출하면 toString()이 생략된다.

대입: array[index] = value; 예) a[1]="B":

배열의 타입을 typeof 로 검사할 수 없다.

Array.isArray([]) :

Array[max index+1]= value; : 자바스크립트는 동적으로 배열의 길이를 조절한다.
