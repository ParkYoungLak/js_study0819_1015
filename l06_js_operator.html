자바스크립트의 숫자와 연산

산술 연산자

operator(연산자) 이름 설명 예시
+ 더하기 두 수를 더한다. 6.8+4.2
- 빼기 두 수를 왼쪽 기준으로 뺀다. 10.5-30
* 곱하기 두 수를 곱한다. 10.3*20
/ 분할 왼쪽 수를 오른쪽 수로 나눈다. 20/3
% 나머지 나누고 남은 나머지 20%3
a**b 거듭제곱 자신의 a를 b번만큼 곱한다. 8**3 Math.pow(8,3)

할당 연산자와 증감 연산자

operator(연산자) 이름 설명 예시
a+=b 추가할당 a=a+b : 자기 자신a에 b를 더한다. x+=10
a-=b 빼기할당 a=a-b : 자기 자신a에 b를 뺀다(뺀 값을 대입한다.). x-=10
a*=b 곱셈 할당 a=a*b : 자기 자신a에 b를 곱한다(곱한 값을 대입한다.). x*=10
a/=b 분할 할당 a=a/b : 자기 자신a에 b로 나눈다(나눈 값을 대입한다.). x/=10
++a; or --a; 증감 및 감소 연산자 a=a+1; or a=a-1; 숫자 앞에 증가 및 감소 연산자가 오면 연산을 먼저 하고 다른 실행을 옮긴다. a(++i); : i를 1증가시키고 a()를 실행하라
a++; or a--; 증감 및 감소 연산자 a=a+1; or a=a-1; 숫자 뒤에 증가 및 감소 연산자가 오면 다른 실행을 하고 연간을 한다. a(i++); : a()를 실행시키고 i를 1증가시켜라.

비교 연산자

참과 거짓을 테스트 하는 비교연산자

operator(연산) 이름 설명 예시
== 등호 a와 b의 값이 같은지 비교 10==(6+4) :
10=="10" :
!= 부등호 a와 b의 값이 같지 않은지 비교 10!=(6+4) :
=== 엄격한 평등 a와 b가 완전히 동일해야 한다. 10===new Number(10) :
10==="10" :
!== 엄격한 비평등 a와 b가 완전히(type까지 ) 동일하지 않은지 비교한다. 10!==new Number(10) :
10!=="10" :
a<b 미만(작다) 왼쪽 값 a가 오른쪽 값 b 보다 작은지 확인. 5<3 : false
a>b 초과(크다) 왼쪽 값 a가 오른쪽 값 b 보다 큰지 확인. 5>3 : true
a<=b 이하 왼쪽 값 a가 오른쪽 값 b 보다 같거나 작은지 확인. 5<=3 : false
a>=b 이상 왼쪽 값 a가 오른쪽 값 b 보다 같거나 큰지 확인. 5>=3 : true

내장 객체 Math의 전역변수와 함수를 알아보자.(Math.PI) :

전역변수 E, PI, LOG*2E, LOG10E,SQRT2,SQRT1_2,LN2...수학에서 사용하는 상수가 존재

Math.method

method 설명 예시 Math.abs() 절대값을 반환. -10의 절대값 Math.abs(-10) Math.abs(10) Math.abs(-10.23) Math.pow(a,b) a의 지수 b(a를 b번 거듭제공) Math.pow(2,8) === 2**8 Math.max(args...) 제공된 숫자들 중 최대값을 반환. Math.max([10,8,20,17,13]) Math.min(args...) 제공된 숫자들 중 최소값을 반환. Math.min([10,8,20,17,13]) Math.min([-10,8,-20,17,-133]) Math.random() 0부터 1사이의 부동소수점 난수를 얻는다. Math.random(): Math.random()*100: /Math.round 반올림. Math.round() 반올림한 정수값을 반환. Math.round(14.5) Math.round(14.435) Math.floor() 내림한 정수값을 반환. Math.floor(14.5) Math.floor(14.435) Math.trunc() 정수만 반환 Math.trunc(14.5) Math.trunc(14.435) Math.sign() 양수인지 음수인지 확인(양수면 1,음수면-1,0이면 0을 반환) Math.sign(-14.5) Math.sign(14.5) Math.sign(0)
