display 속성으로 노드의 레이아웃 제어

display block

항상 새로운 라인으로 시작하고 가로폭 전체를 차지한다.(width:100% = default)

  • 너비(width) 가 지정될 때 마진(margin)이 남은 가로폭을 채운다.
  • margin-left: 0 , margin-right: auto; 로 블럭이 왼쪽 정렬한다.
  • 보통 block은 inline 요소를 포함하기 위해 지정한다.
  • box model 의 속성을 모두 포함한다.
    • width: 넓이, max-width:최대넓이, min-width:최소넓이
    • height: 높이, max-height: 최대높이, min-height: 최소높이
    • padding: 컨텐츠와 보더 사이 여백( top right botton left).
    • padding-top= padding: top 0 0;
    • padding-right= padding: 0 right 0 0;
    • padding-bottom= padding: 0 0 bottom 0;
    • padding-left= padding: 0 0 0 left;
    • border: box 의 윤곽선(두께 스타일 색)( top right botton left ).
    • margin: 컨텐츠와 보더 사이 여백( top right botton left).

display inline

줄을 바꾸지 않고 다른 요소와 한행에 위치한다. (width:fit-content default)

  • box 모델 중에 width, height, ,margin-top, margin-botton 을 지정할 수 없다.
  • height 대신 line-height 로 상하 여백을 지정한다.
  • inline 의 스타일을 변경하거나 쓰임을 명시하기 위해 작성한다.

display inline-block

inline 과 같이 줄을 바꾸지 않고 다른 요소와 한행에 위치하는데 block 처럼 box 모델 모두를 지정할 수 있다.(width: fit-content default)

  • 요소 내부 컨텐츠의 뒤쪽 여백을 4px 로 자동 지정한다.
  • 부모 요소의 컨텐츠(텍스트) 세로 정렬을 inline-block 내부의 컨텐츠 세로정렬이 같이 된다.
  • vertical-align 으로 컨텐츠의 세로정렬을 한다.

display none

  • display none 은 해당 요소를 화면에 표시하지 않는다.
  • 해당 요소는 노드 개체가 삭제되지는 않는다.
  • display 는 애니메이션 대상에서 제외된다.

visibility 요소를 보이게 할건지를 결정한다.

visibility hidden

  • visibility : hidden 은 해당 요소를 화면에 보이지 않게 한다.
    • 해당 요소의 box 속성을 그대로 남기고 보이지만 않게된다.
  • visibility: visible 은 default값으로 해당 요소를 보이게 한다.
  • visibility 는 애니메이션 대상에 포함되어 있지만 오작동한다.

opacity 0= hidden

  • opacity : 0 은 visibility : hidden 과 거의 동일하게 작동한다.
  • opacity는 요소의 투명도를 조절할 수 있다.
  • rgba로 요소의 배경투명도를 변경하는 것과는 차이가 있다.

transform scale(x,y)

  • style.transform 은 노드 개체를 2d ,3d로 변환해서 회전,배율,이동,기울기 등을 변경한다.
  • transform 을 적용한 노드는 실제크기가 변경되는 것이 아니라 스냅샷 같은 것이 변경되서 화면에 표현되기 때문에, 다른 개체를 밀어내거나 당기지 않는다
  • transform 은 애니메이션을 적용하기에 최적의 속성이다.
