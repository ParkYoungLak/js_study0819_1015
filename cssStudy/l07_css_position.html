position 요소,노드 개체의 위치를 정의한다.(static, relative, absolute, fixed, sticky)

position : static (기본위치 default)

모든 요소, 노드 객체는 위치를 static 으로 정의한다.

  • 요소의 배치는 순차적이며 (위에서부터 아래로) 왼쪽 에서부터 오른쪽 으로 배치된다.
  • 자식요소는 부모요소의 위치를 기준으로 배치된다.
  • top, botton, left, right 를 사용할 수 없다.

position : relative (상대적 위치)

자식박스를 상대적 위치로 왼쪽 위 100px 씩 이동.

  • 상대적 위치 relative는 정적위치 static 을 기준으로 노드를 이동시킨다.
  • 상대적 위치로 이동한 노드 객체는 다른 노드에 영향을 주지 않는다.

position absolute(절대적 위치)

절대적 박스
부모 노드가 static 위치일 때
절대적 위치의 기준은 root노드인 body이다.

position absolute(절대적 위치)

절대적 박스
부모 노드 중에 relative 위치가 있을 때
절대적 위치의 기준은 relative위치를 가진 부모노드 기준.
  • absolute는 부모 요소중 relative,absolute,fixed가 정의되어 있으면 그 부모를 기준으로 위치를 이동한다.
  • 만약 부모노드 중에 relative,absolute,fixed가 없다면 body를 기준으로 위치가 고정된다.
  • 부유객체로 다른 요소를 밀거나 당기지 않고 덮어쓰는 형태로 존재한다.
  • absolute로 정의된 노드는 넓이가 content에 맞게 지정된다. 때문에 넓이를 지정하는 것이 좋다.
