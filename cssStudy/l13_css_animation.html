animation은 javascript 의 setInterval로 만드는 동작을 간단하게 작성할 수 있다.

  • animation: 모든 프로퍼티를 작성. (shorthane syntax)
  • animation-name: @keyframes 이름을 지정 (이벤트로 애니메이션을 등록할 요소에 animation-name을 추가하면 된다.=> 만약 동작하지 않으면 이름이 정의된 클래스를 추가하면 된다.
  • @keyframes animation-name: 애니메이션이 동작하는 단계를 나눠서 설정.
  • animation-duration: 한싸이클의 애니메이션이 쇼요되는 시간( s or ms default 0)
  • animation-timing-function: 애니메이션이 속도를 나타내는 함수(default ease)
  • animation-delay: 요소가 시작될 시점(-delay: 애니메이션이 동작된 미래를 나타낸다.)
  • animation-iteration-count : 애니메이션의 재생 횟수.(default:1,infinite)
  • animation-play-state: 애니메이션의 재생상태를 지정(default:running stop:paused)
  • animation-direction: {normal: from0~to100%, reverse: to~from, alternative: 홀수번째는 normal 짝수번쨰는 reverse, alternative-reverse: alternative의 반대.}
  • animation-fill-mode:
    • none: 시작점에서 keyframes 0%에 적용된 스타일을 적용하지 않고 대기하다.
      애니메이션이 종료되면 원래 상태로 되돌리고 종료한다.
    • forwards: none과 시작은 같고 애니메이션이 종료되면 그 상태 그대로 종료한다.
    • backwoards: 시작점에서 keyframes에 적용된 스타일을 적용하고 대기하다.
    • both: 시작은 backwoards와 같고 종효는 forwards와 같다.
