자바스크립트의 루프와 반복.

for, for in, for of, while, dowhile 반복문(iteration statements)이 있다.

동일한 명령을 일정 횟수만큼 방복하기 위해 작성되는 문법.대체로 검사식에 많이 사용된다.

for 반복문

for(초기문; 조건문; 증감문){반복실행} : for는 조건이 true일 때 반복실행한다.

  • [11,2,20,3,13,1,0,-20]의 max값 찾기

for in 반복문

for(key in iteration Object){} : key 를 가진 객체가 key 를 받을수 없을 떄까지 반복을 실행

JSON(key:value, key,value) =>key를 반환, array[a,c,b] => index를 반환.

  • [a,b,c]
  • {name:"최경민", age:36, email:"artistckm86@gmail.com"}

for of 반복문

for( object of iterable Object(하지만 주로 배열))

반복 가능한 데이터 타입의 객체에서 더 이상 인자가 나오지 않을 떄까지 반복 실행

(iterationObject).next() : next() 는 포인트가 객체 밖에 있다가 내용물이 있으면 true 를 반환하고 내용물로 포인트를 옮긴다.그리고 내용물도 반환한다.만약 값이 더이상 나오지 않으면 false를 반환.

배열,Map(JSON과 비슷),Set(배열인데 복수의 값을 제거),문자열

key 값을 알아내기 힘들다.

  • ["a","b","c"];
  • {black:["black,"white"],default:["white","black"],darkgray:["darkgray",bisque"],pink["lavender","indigo"]}
